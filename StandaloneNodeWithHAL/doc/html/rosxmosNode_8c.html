This function copies a string into a buffer. The starting point inside the string is given with the index pointer, which is also increased. This is used to be able to add multiple strings behind each other in the same buffer. Be sure your buffer is big enough, there are no checks for the buffer size for increasing performance!

Parameters:
indexThe current index
bufferThe buffer to write the string to.
strThe terminated string to write into the buffer
